    Create `app/services/git_policy_provider.py` with:

    1. `GitPolicyProvider` class:
       - `__init__(repo_url: str, branch: str = "main", local_path: str = ".policy_repo", fallback_dir: str = "app/policies")`:
         - Store config, create asyncio.Lock, initialize empty cache
       - `_ensure_repo()` async method (uses asyncio.to_thread for blocking Git operations):
         - If local_path exists and is a git repo: `repo.remotes.origin.fetch()`
         - Else: `git.Repo.clone_from(repo_url, local_path, branch=branch)`
         - Uses `self._lock` to prevent concurrent operations
       - `get_policies()` async method:
         - If `repo_url` is empty: fall back to local `fallback_dir`, load via PolicyEngine.load_from_directory(), return `(policies_dict, "local")`
         - Call `_ensure_repo()`
         - Get `head_sha = repo.head.commit.hexsha`
         - If `head_sha` in cache: return cached result
         - Read all YAML files under `policies/global/`, `policies/projects/`, `policies/temporary/` from `repo.head.commit.tree`
         - Parse each YAML file content
         - Cache by SHA: `self._cache[head_sha] = result`
         - Return `(policies_by_layer, head_sha)`
       - `get_policies_for_project(project_id)` async method:
         - Get all policies via `get_policies()`
         - Return global + project-specific + temporary policies combined
       - `get_policies_at_commit(commit_sha)` async method:
         - Read policies from `repo.commit(commit_sha).tree` (not HEAD)
         - Useful for audit trail verification
       - `_read_yaml_from_tree(tree, path)` method:
         - Navigate tree[path_parts], read blob data, parse YAML
         - Return parsed dict or None if file not found
