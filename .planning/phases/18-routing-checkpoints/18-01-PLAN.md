# Phase 18 Plan 01: Approval Router with Priority Queues + Batch Approval

## Objective

Add priority-ordered approval queue and batch approve/reject operations so reviewers can efficiently handle bursts of review requests with urgent items surfaced first.

## Context

- @app/core/policy.py — Policy engine evaluates reviews and routes to APPROVING for HUMAN/AI_AUDIT
- @app/api/v1/reviews.py — List reviews endpoint currently sorts by id-desc only
- @app/api/v1/actions.py — Single approve/reject endpoints (one review at a time)
- @app/models/schema.py — Review model has priority field (low/normal/high/critical)
- @app/models/schemas.py — Pydantic request/response schemas
- @app/core/state_machine.py — transition_state handles state transitions with optimistic locking
- @.planning/phases/18-routing-checkpoints/18-RESEARCH.md — Design decisions

## Success Criteria

1. `GET /api/v1/reviews/?status=APPROVING&sort=priority` returns reviews ordered by priority weight (critical first), then by created_at (oldest first within same priority)
2. `POST /api/v1/reviews/batch/approve` processes multiple reviews, returns 207 Multi-Status with per-item success/failure
3. `POST /api/v1/reviews/batch/reject` processes multiple reviews with mandatory reason, returns 207 Multi-Status
4. Each batch item creates its own audit trail entry
5. Batch operations support both JWT auth and one-time review tokens
6. Default sort behavior (id-desc) unchanged for backward compatibility

### Task 1: Create ApprovalRouter service module
type="auto"

**Done when:**
- `app/services/__init__.py` directory created
- `app/services/approval_router.py` exists with:
  - `PRIORITY_WEIGHT` mapping: critical=4, high=3, normal=2, low=1

**Implementation:**
```python
# app/services/approval_router.py
from sqlalchemy import case, select
from sqlalchemy.ext.asyncio import AsyncSession
from app.models.schema import Review
from app.models.schemas import ReviewState

PRIORITY_WEIGHT = {"critical": 4, "high": 3, "normal": 2, "low": 1}

def build_approval_queue_query(limit: int = 50, offset: int = 0):
    """Build a priority-ordered query for APPROVING reviews."""
    priority_order = case(
        *[(Review.priority == p, w) for p, w in sorted(PRIORITY_WEIGHT.items(), key=lambda x: -x[1])],
        else_=0,
    )
    return (
        select(Review)
        .where(Review.state == ReviewState.APPROVING.value)
        .order_by(priority_order.desc(), Review.created_at.asc())
        .limit(limit)
        .offset(offset)
    )

async def get_priority_sorted_reviews(session: AsyncSession, limit: int = 50, offset: int = 0) -> list[Review]:
    """Fetch APPROVING reviews sorted by priority (critical first), then creation time."""
    query = build_approval_queue_query(limit, offset)
    result = await session.execute(query)
    return list(result.scalars().all())
```
