    1. **Role enum** (add at top, after imports):
       ```python
       import enum

       class Role(str, enum.Enum):
           ADMIN = "admin"
           REVIEWER = "reviewer"
           AUDITOR = "auditor"
           AI_SERVICE = "ai_service"
       ```

    2. **Extend `create_jwt`** to accept optional `role` parameter:
       - Change signature: `def create_jwt(client_id: str, jwt_secret: str, expires_minutes: int = 15, role: str = "reviewer") -> str`
       - Add `"role": role` to payload dict.
       - Default role is "reviewer" (backward compatible -- existing callers that don't pass role get reviewer).

    3. **Create `require_role` FastAPI dependency**:
       ```python
       def require_role(*allowed_roles: str):
           """FastAPI dependency factory: validates JWT and checks role claim."""
           async def _check_role(
               credentials: HTTPAuthorizationCredentials = Depends(security),
               settings: Settings = Depends(get_settings),
           ) -> dict:
               try:
                   payload = decode_jwt(credentials.credentials, settings.jwt_secret)
               except AuthenticationError as e:
                   if "expired" in str(e).lower():
                       raise HTTPException(status_code=401, detail="Token expired")
                   raise HTTPException(status_code=401, detail="Invalid token")
               user_role = payload.get("role", "reviewer")
               if user_role not in allowed_roles:
                   raise HTTPException(status_code=403, detail=f"Role '{user_role}' not authorized. Required: {list(allowed_roles)}")
               return payload
           return _check_role
       ```

    4. **Create convenience dependencies** for common role combinations:
       - `require_admin = require_role(Role.ADMIN)` -- admin only
       - `require_reviewer = require_role(Role.REVIEWER, Role.ADMIN)` -- reviewer + admin
       - `require_auditor = require_role(Role.AUDITOR, Role.ADMIN)` -- auditor + admin (read-only)
       - `require_ai_service = require_role(Role.AI_SERVICE)` -- ai_service only
       - `require_any_role = require_role(Role.ADMIN, Role.REVIEWER, Role.AUDITOR, Role.AI_SERVICE)` -- any authenticated role

    These are module-level variables, created after `security` is defined (line 46 in current file).

    5. Keep all existing functions unchanged in behavior (`require_jwt`, `get_current_client`, one-time tokens, capability tokens). The `role` field is additive.

          """Return dict with 'client' and 'role' from JWT cookie."""
          if not access_token:

          except AuthenticationError:
