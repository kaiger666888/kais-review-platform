<objective>
Build the core aggregation pipeline services: typed event protocol, topology collapser, progressive fill engine, and aggregator orchestrator.

Purpose: These services form the heart of Shot Card aggregation -- they take raw OpenClaw node completion events, map them to the correct Shot Card JSONB fields via the topology collapser, incrementally merge data via the progressive fill engine, and coordinate the full pipeline via the aggregator. They handle out-of-order completion, concurrent creation race conditions, and min_audit_set readiness evaluation.

Output: Four new modules under app/core/ and app/services/ that are fully self-contained and importable by downstream plans (API endpoints, arq tasks).
</objective>

4. **app/services/progressive_fill.py** -- ProgressiveFillEngine class:
   - `async fill(shot_id: str, target_column: str, merge_data: dict) -> ShotCard | None` method:
     - Uses `async_session_factory()` to get a session.
     - SELECT ShotCard WHERE shot_id == shot_id.
     - If not found, return None (aggregator handles creation).
     - Get current JSONB value via `getattr(shot_card, target_column)` -- default to `{}` if None.
     - Deep merge new data: `merged = _deep_merge(current, merge_data)`.
     - `setattr(shot_card, target_column, merged)`.
     - **CRITICAL**: Call `flag_modified(shot_card, target_column)` to ensure SQLAlchemy detects the mutation.
     - Commit and refresh.
     - Log with structlog: "progressive_fill" with shot_id, column, fields_updated.
     - Return the updated ShotCard.
   - `_deep_merge(base: dict, override: dict) -> dict` static method:
     - Recursively merge override into base.
     - Dict values merged recursively, non-dict values in override replace base values.
     - Order-agnostic: works regardless of which fields arrive first.

     - `audio_bundle`: requires (`bgm_prompt` OR `sfx_prompt`) AND `status != "pending"`.

The `ShotCardAggregator` class:

- `async handle_node_completion(event: dict) -> dict` -- the main pipeline:
  1. Extract `shot_id` and `node_type` from event.
  2. Call `self.collapser.collapse(node_type, event["node_output"])` to get target_column and merge_data.
  3. Call `await self._ensure_shot_card(event)` to get or create the ShotCard.
     - If None returned, return `{"status": "error", "reason": "failed_to_create_shot_card"}`.
  4. Call `await self.filler.fill(shot_id, target_column, merge_data)`.
     - If None returned, return `{"status": "error", "reason": "fill_failed"}`.
  5. Call `self.filler.check_min_audit_set(shot_card)` and `self.filler.check_bundle_complete(shot_card, target_column)`.
  6. Call `await self._emit_events(shot_card, target_column, bundle_complete, min_audit_satisfied)`.
  7. Return `{"status": "ok", "shot_card_id": ..., "shot_id": ..., "updated_column": ..., "bundle_complete": ..., "min_audit_satisfied": ...}`.

- `async _ensure_shot_card(event: dict) -> ShotCard | None` -- get or create:
  - Import `ShotCard` from `app/models/shot_card.py`, `select` from sqlalchemy, `async_session_factory` from `app/core/database.py`, `IntegrityError` from `sqlalchemy.exc`.
  - Open async session, SELECT ShotCard WHERE shot_id == event["shot_id"].
  - If found, return it.
  - If not found, create new ShotCard with:
    - `shot_id=event["shot_id"]`
    - `project_id=event["project_id"]`
    - `narrative_context=event.get("narrative_context", {})`
    - `min_audit_set=["visual_bundle"]` (default)
    - `workflow_version=event.get("provenance", {}).get("workflow_version")`
    - `policy_commit_sha=event.get("provenance", {}).get("policy_commit_sha")`
    - `execution_id=event.get("execution_id")`

    - On IntegrityError: rollback, re-query the existing card by shot_id, return it.
