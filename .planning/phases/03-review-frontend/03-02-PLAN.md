<objective>
Wire SSE real-time updates into the dashboard using HTMX 2.0 SSE extension.

Purpose: When a review status changes (approve/reject/new submission), the dashboard updates automatically without page refresh. This addresses the critical HTMX 2.0 SSE migration — the old hx-sse attribute does NOT work; must use htmx-ext-sse extension with sse-connect/sse-swap attributes.

Output: Live-updating dashboard with SSE-driven review list refresh and "new reviews" banner.

CRITICAL RESEARCH FINDING: HTMX 2.0 requires separate extension htmx-ext-sse@2.2.4 with sse-connect/sse-swap attributes. The old hx-sse syntax silently fails. SSE endpoint must accept cookie auth since EventSource API cannot set Bearer headers.
</objective>
