       - `create_connection() -> asyncio.Queue`: creates `asyncio.Queue(maxsize=100)`, adds to `self._connections`, logs via structlog `"sse_connected"` with `total_connections=len(self._connections)`, returns the queue
       - `remove_connection(queue)`: calls `self._connections.discard(queue)`, logs `"sse_disconnected"` with `total_connections=len(self._connections)`
       - `async broadcast(event_data: dict)`: iterates `self._connections`, calls `queue.put_nowait(event_data)` wrapped in try/except asyncio.QueueFull -- on QueueFull add queue to a `disconnected` list, then remove all disconnected queues. If `len(self._connections) > 50` log a warning via structlog.
       - `connection_count` property returns `len(self._connections)`
    2. `event_manager = EventManager()` -- module-level singleton instance

       - Returns `EventSourceResponse`
       - Calls `event_manager.create_connection()` to get a queue

         - `while True` loop:

       - In `finally`: `event_manager.remove_connection(queue)`
