**Class TestWebhookDelivery** -- Tests the deliver_webhook task function with mock httpx client:

These tests follow the existing pattern from tests/test_webhooks.py but are organized as integration tests in the integration package, using the same db_session and mock patterns.

   - Verify no Retry exception is raised (delivery gives up)
